# Merchant Churn Risk Dashboard – Professional Product Prompt

## Objective

Build a modern, enterprise-grade web dashboard that helps Customer Success and Sales teams proactively identify merchants who are likely to churn and recommends the best next action.

The application should look like a premium SaaS platform with clean UI, smooth interactions, excellent spacing, and responsive design.
